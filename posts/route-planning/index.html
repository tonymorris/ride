Route Planning

Posted on June 7, 2013, in Route Planning

In the repository, you will find a Haskell script for generating a URL containing waypoints. The URL points to the Open Source Routing Machine (OSRM), which then produces a route that visits each waypoint.

There is a problem in that some of the OpenStreetMap (OSM) tracks are not marked as having vehicular access. This means that OSRM will not use them for routing. This is especially prevalent in western Queensland and the Simpson desert. Therefore, although I have created many route proposals, I am waiting for the OSM map to update on OSRM so that it routes along vehicle tracks.

If you would like to propose a route, please use the OSRM website, generate a URL and submit it to the mailing list. You can also do this to modify an existing route proposal.
